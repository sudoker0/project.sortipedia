# Sortipedia

A clone of the Sound of Sorting, written in TypeScript and made because I was bored.

## Dependecies

NodeJS

## Build

```
npm i # to install packages
node build.mjs # to compile
```

## Serve

The compiled website is provided at `dist/`, you may use any software to serve from the directory.

For example, you can use `http.server` library in Python to serve the website as follows:

```
python -m http.server --directory dist/
```
